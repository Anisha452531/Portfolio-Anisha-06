Bank Churn Prediction using ML algorithms

Objective: Predict whether a bank customer is likely to leave (churn) or stay, helping the bank retain valuable customers.

Dataset: Historical customer data including demographics, account information, activity metrics, and the target variable: Churn (Yes/No) with probability percentage.

Techniques & Tools Used:

  • Python for data analysis and modeling
  • XGBoost for predictive modeling
  • SMOTE to handle class imbalance
  • SHAP for interpreting model predictions and identifying key factors influencing churn
